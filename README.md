# storage
It is a photo storage site built with Pocketbase and Svelte.

## 🛠 Installation (Website)
- First download the project.
- Secondly, we set up the project
`npm i` & `pnpm i`

- Thirdly, run the project.
`npm dev` & `pnpm dev`

## 🛠 Installation (Pocketbase)
- Install your Pocketbase project and create an administrator account.
- Take the link to the Pocketbase server you created and paste it into the .env file in the project.
![Image](https://cdn.discordapp.com/attachments/1136591322208272385/1151877645139710022/Code_WO1P6Nt6xp.png)

- Create a user in the "users" collection.
![Image](https://i.imgur.com/EzVXam9.png)

- Fill in all the blanks according to your own wishes and save.
![Image](https://cdn.discordapp.com/attachments/1136591322208272385/1151873978374311997/brave_y7wVJrkKeJ.png)

- Press the "New Collection" button on the side panel.
![Image](https://cdn.discordapp.com/attachments/1136591322208272385/1151876324659560468/brave_lNogYUz8Om.png)
